# Heart Disease Dataset Pipeline
### Multi-Model Explainable AI System — Medical Decision Support

**BS Artificial Intelligence Semester Project**

---

## Project Overview

This pipeline trains and evaluates Logistic Regression and Decision Tree classifiers
on the UCI / Kaggle Heart Disease dataset. It produces LIME-ready serialised models
and a StandardScaler for use in the downstream explainability module.

## Setup & Execution

```bash

pip install -r requirements.txt

```

Run all cells top-to-bottom. The notebook is self-contained and reproducible.

## LIME Integration

The saved `logistic_regression_tuned.pkl` and `standard_scaler.pkl` are the primary
artefacts for downstream LIME explainability. Load them as follows:

```python
import joblib
from pathlib import Path

scaler    = joblib.load('models/standard_scaler.pkl')
model     = joblib.load('models/logistic_regression_tuned.pkl')

# For LIME: wrap prediction in a pipeline-compatible function
import lime.lime_tabular
explainer = lime.lime_tabular.LimeTabularExplainer(
    X_train_scaled.values,
    feature_names=feature_names,
    class_names=['No Disease', 'Disease'],

)
```
